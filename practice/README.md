# Practice Exercises

Welcome to the practice exercises folder! This directory contains hands-on exercises and learning materials for mastering AI development tools.

## What's Here?

All practice materials are organized in one place to make it easy to find exercises, experiment with AI tools, and contribute your own learning resources.

### Practice Folders

- **sandbox/** - AI model comparison experiments with evaluation rubrics
- **cline-scenarios/** - Cline plan & execute workspace exercises with acceptance criteria
- **copilot-katas/** - GitHub Copilot refactoring drills and coding challenges
- **mcp/** - MCP server configurations, testing guides, and smoke tests
- **safety-cases/** - Prompt safety & RAI red team exercises for understanding AI security
- **.claude/** - Claude Code CLI task files for terminal-based development

### Reference Files

- **agents.md** - Build, run, and test commands for AI coding agents

## How to Use

1. **Interactive Practice**: Visit the AI Sandbox tab in the application for guided exercises with copy-to-try prompts
2. **Repository Practice**: Each folder contains detailed README files with exercises you can complete by creating branches and opening PRs
3. **Learn by Doing**: Follow the "Objective → Context → Steps → Verify" pattern in each exercise

## Getting Started

New to this repository? Start here:

1. Read `agents.md` to understand how to build and test the project
2. Explore `cline-scenarios/` for your first guided Cline exercise
3. Try `copilot-katas/` to practice with GitHub Copilot
4. Experiment with `sandbox/` to compare different AI models

## Contributing

Want to add your own exercises?

1. Create a branch: `git checkout -b add-exercise-name`
2. Add your exercise to the appropriate folder
3. Follow the "Objective → Context → Steps → Verify" pattern
4. Include sample data, stubs, and expected outputs
5. Open a PR with a description of the exercise and lessons learned

See individual folder READMEs for specific contribution guidelines.
